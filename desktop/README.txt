================================
Auto Mail Pilot v0.0
================================

メール自動送信デスクトップアプリケーション

【使い方】

1. config.json.example を config.json にリネームしてください
2. config.json 内の gas_url に、GAS WebアプリのURLを設定してください
3. auto-mail-pilot_v0.0.exe を起動してください

【初回設定】

1. アプリを起動
2. 「設定」タブを開く
3. GAS WebアプリURLを入力
4. 「接続テスト」ボタンで接続を確認
5. 署名を設定（任意）
6. 「設定を保存」ボタンで保存

【機能】

- 宛先マスタ（スプレッドシート）からの宛先選択
- テンプレートの自動適用
- ファイル名から宛先・テンプレートの自動選択
- 複数宛先対応（最大3件）
- 添付ファイル送信
- 署名自動挿入

【動作環境】

- Windows 10/11
- macOS
- Linux

【トラブルシューティング】

Q: 接続エラーが出る
A: GAS URLが正しいか確認してください。
   GASのデプロイ設定で「アクセスできるユーザー」が「全員」になっているか確認してください。

Q: 宛先やテンプレートが表示されない
A: スプレッドシートの構成が正しいか確認してください。
   「テンプレート」「宛先リスト」シートが存在するか確認してください。

【ログファイル】

実行ファイルと同じフォルダの logs/ 配下にログが出力されます。
エラー発生時はログを確認してください。
